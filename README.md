# 🧠 AI-Powered Medical Research Assistant (MERN + LLM)

A full-stack **AI-powered medical research assistant prototype** built using the **MERN stack** and a **custom open-source LLM**.

This system goes beyond a chatbot — it acts as a **research + reasoning engine** that understands user context, retrieves credible medical data, and delivers structured, source-backed responses.

---

## 🚀 Features

- 🧠 **Context-Aware Query Understanding**
  - Interprets user queries with medical context awareness

- 🔎 **Research Retrieval System**
  - Fetches high-quality data from trusted medical sources (e.g., PubMed, research APIs)

- 🤖 **LLM-Based Reasoning**
  - Uses an open-source LLM to analyze and synthesize retrieved data

- 📊 **Structured Responses**
  - Outputs:
    - Summary
    - Key insights
    - Risks / considerations
    - References (source-backed)

- 🔐 **User Session Handling**
  - Maintains conversational context per user
